# ペルソナ

## ペルソナ1: 一般利用者「みなと」

- **役割**: 画像の内容を文章で素早く知りたい利用者
- **技術レベル**: 一般的なWeb操作に慣れている
- **目的**: 手元の画像が何を写しているかを日本語で把握する
- **ニーズ**: 簡単な操作、分かりやすい説明、詳しさの調整
- **アクセシビリティ**: キーボード操作とスクリーンリーダーを利用する場合がある

## ペルソナ2: 開発者「そら」

- **役割**: OpenAIによる画像説明機能を試すエンジニア
- **技術レベル**: 高い
- **目的**: ローカルで動かして挙動、プロンプト、モデルを検証する
- **ニーズ**: 明確なエラー、設定可能なモデル、安全なAPIキー管理
- **アクセシビリティ**: 標準的な要件

## ペルソナと機能の対応

| ペルソナ | 主要機能                                             |
| -------- | ---------------------------------------------------- |
| みなと   | 画像選択、詳細度選択、説明表示、コピー               |
| そら     | モデル設定、エラー詳細区分、ヘルスチェック、テスト   |
