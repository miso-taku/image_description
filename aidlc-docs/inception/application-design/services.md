# サービス定義

## S-1 画像説明サービス (Description Service)

- **責務**: 検証済み画像と詳細度から日本語説明を生成するオーケストレーション。
- **手順**:
  1. 詳細度に応じたプロンプトを構築する。
  2. PydanticAI Agentへ画像（BinaryContent）とプロンプトを渡す。
  3. 構造化出力を検証し `DescriptionResponse` を返す。
  4. 外部エラーをドメイン例外へ変換する。
- **非機能**: タイムアウト、機密情報を含まないログ、失敗時の安全なエラー。

## S-2 検証サービス (Validation Service)

- **責務**: 入力画像の形式・サイズ・シグネチャ検証。
- **特徴**: 純粋関数で副作用なし。単体テストとPBTの主対象。

## オーケストレーション

API層がリクエストごとにリクエストIDを生成し、検証サービス、説明サービスの順に呼び出す。いずれの段階の失敗もグローバル例外ハンドラーが `ErrorResponse` へ整形する。
